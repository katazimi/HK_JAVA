메일 수신 여부


라디오 버튼은 하나만 체크되고 다시 취소되지 않는다. 데이터 전송시 체크된 요소만 값이 전송된다.

관심 분야


체크박스는 여러개를 동시에 체크할 수 있다. 취소도 가능하다. 데이터 전송시 체크된 요소만 값이 전송된다.

파일 업로드


select 요소에 name 속성 정의, option 요소에 value 속성정의 데이터 전송시 선택된 값이 정의된다. (여러개를 선택할 수도 있긴함.. multiple 속성 사용)

form 요소에 enctype="multipart/form-date" 속성을 정의해야함
